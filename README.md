# Kindlimann Bestellliste

Arbeitsmappe für die Sammelbestellung an Kindlimann.

## Wichtige Dateien

- `Kindlimann Bestellliste.xlsm` - fertige Excel-Datei für die Nutzung
- `Bestellt` - Zielordner für erzeugte PDFs
- `_Entwicklung` - Generator, Konfiguration und technische Notizen

## Hinweise

- Die Excel-Datei muss mit Excel Desktop geöffnet werden.
- Makros müssen aktiviert sein, damit Buttons, PDF-Export und E-Mail-Vorbereitung funktionieren.
- Exportierte PDFs im Ordner `Bestellt` werden nicht in GitHub gespeichert.
- Änderungen an Profilen, Mailtext, Rändern und Abständen zuerst in `_Entwicklung\Config\kindlimann.settings.psd1` pflegen.
